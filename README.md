# Dijital Ansiklopedi

Modern mobil ansiklopedi uygulaması

## Kurulum ve Çalıştırma

### Gereksinimler
- Flutter SDK (3.0.0 veya üzeri)
- Android Studio veya Xcode (platform bazında)

### Adımlar

1. **Flutter SDK Kurulumu**
   - Flutter'ı indirin: https://flutter.dev/docs/get-started/install
   - PATH'e ekleyin

2. **Projeyi Çalıştırma**
   ```bash
   # Proje klasörüne gidin
   cd dijital_ansiklopedi

   # Bağımlılıkları yükleyin
   flutter pub get

   # Uygulamayı çalıştırın
   flutter run
   ```

3. **Platform Seçenekleri**
   - Android: Emülatör veya fiziksel cihaz
   - iOS: Simulator veya fiziksel cihaz (macOS gerekli)
   - Web: `flutter run -d chrome`
